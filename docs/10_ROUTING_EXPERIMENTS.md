# Routing Experiments
**Proposed by Q, July 4, 2026: "Right now we have intuition. Not
routing data. If you write routing rules today, they're likely to
become folklore. After 30-50 real cases, the routing policy will
almost write itself."**

This file is deliberately empty of rules. It exists to accumulate real
Observer Council sessions until an actual topic-adaptive routing
policy can be derived from evidence rather than guessed in advance —
same discipline as everything else in this project: measure before
architecture.

**Do not add routing rules to this file.** When enough entries exist
below to see a real pattern, that pattern becomes a Working Hypothesis
in `05_WORKING_HYPOTHESES.md` — proposed from evidence, not before it.

```
Date:
Claim/Question:
Observers used:
Disagreements:
Outcome (synthesis reached, or unresolved):
Hindsight (add later — did the outcome hold up?):
```
