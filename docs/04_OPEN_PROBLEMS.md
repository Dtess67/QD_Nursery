# Open Problems
**Questions nobody has answered yet. Not hypotheses — no proposed
answer exists for these.**

- Should Layer 3 (Behavioral Governance) stay named specifically for
  human safety, or broaden to cover animals, bystanders' property, and
  other embodied agents? Flagged specifically: broadening it would move
  bystander property damage from Layer 2 (acceptable scar) into Layer 3
  (non-bypassable) without that reclassification getting the scrutiny
  the human-safety line received. Proposed by Q, July 1, not adopted.
- Exact mechanism for how Layer 3 determines "a person may plausibly be
  present" from sensor data — only the requirement is settled
  (persistent belief across observations, informed by the Uber/Herzberg
  case), not the architecture that does it.
- Full specification of Rover's physical self-model, sensor fusion, and
  continuous self-model — per Q, intentionally kept unhardened while
  understanding develops. Status: specified but not built.
- Whether/how to disclose retrieval-result variance inside a verdict
  itself, so a verdict's evidence set is understood as time-bound
  rather than eternal.
